Humu Slides – AI PPT Maker

Type your topic. Get your slides — instantly.

Quick Start (No Backend)

  1. Unzip this folder.
  2. Open index.html or host on GitHub Pages / Netlify / Vercel.
  3. Generate → Edit → Download PPTX.

AI Integration

Implement a serverless endpoint at /api/generate and set AI_ENDPOINT in app.js.

Tech

  • Tailwind (CDN)
  • PptxGenJS
  • Vanilla JS
